Hangman Code

The first thing to think about is the structure of the program. We are going to need these functions:

  • A main function that runs everything
  • A function to pick a random word from a list of words
  • A function that keeps track of what letters we've picked and how much of the word has been figured out
  • A function that draws the hanged man - or maybe several functions, one for each body part
