Intro

We are going to start by just typing some basic Python commands into the interpreter window. We are going to use an online Python environment called Trinket.

Trinkets can be embedded into web pages, like this one. If you want to make your own Trinkets later, the web page is here:

https://trinket.io

for now, let's use the one embedded here.

Type some of these commands - it doesn't matter if you understand everything, it is just to get used to typing python. After typing each command, hit the RUN button

print "hello world"

print 10 + 5

my_name = "" (type your name between the quotation marks)

print my_name

Math

Python can be used like a calculator, and "knows" a lot of math. Look at the code in the Trinket window below. Hit the RUN button and see what happens. One of the really great things about Python, though, is that there are many libraries that can be imported and used. Libraries are bits of code that someone else has written that we can use. If you ever want to do somethign in Python, I suggest searching for libraries that already do what you want to do!

We are going to want to pick a random number, and there is a library called random that we can use. To use a library, you first have to import it.

Delete the code in the trinket above and then type:

import random

into the Trinket window. Now we can use a function in the random library called randrange. randrange takes a number as input, and returns a random number between zero and the number you have given it. Type:

print random.randrange(100)

in your Trinket, and hit run a few times. What happens?
