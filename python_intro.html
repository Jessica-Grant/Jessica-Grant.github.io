One of the most important features of any programming language is its ability to manipulate variables. A variable is just a name that refers to a value; you can think of a variable as a box that stores a piece of data.

In Python, the basic data types are strings and numbers. There are two types of numbers: integers (both positive and negative) and floats (fractional numbers with a decimal point).

Above, you used the varaiable named x to store the value 15 and the variable my_name to store a string

Math

One of the really great things about Python, though, is that there are many libraries that can be imported and used. Libraries are bits of code that someone else has written that we can use. If you ever want to do somethign in Python, I suggest searching for libraries that already do what you want to do!

We are going to want to pick a random number, and there is a library called random that we can use. To use a library, you first have to import it.

Delete the code in the trinket above and then type:

import random

into the Trinket window. Now we can use a function in the random library called randrange. randrange takes a number as input, and returns a random number between zero and the number you have given it. Type:

print random.randrange(100)

in your Trinket, and hit run a few times. What happens?

Strings

Python thinks of words as strings made up of characters. They are defined with quotation marks, like these:

Notice that the spaces are considered characters, too

If you contniue in Python, you will see that there are lots of things you can do with strings. One feature we will use to make our hangman game, is indexing.

Each position in the string has an index, starting with zero.

In the Trinket above, type

print my_string[0] # this is the first character in the string
print my_string[1:8] # this is a 'slice' of the string, from the second character ([1]) to just before but not including the eighth character
print my_string[1:8:2] # this is a 'slice' of the string, as above, but taking every second character
print my_string[-1] # negative indices start from the end, so -1 is the last character

We can also add strings or pices of strings together. Try this
new_string = my_string[:7] + ' not ' + my_string[8:]
print new_string

Lists

Lists are another way to store data. They are stored inside square brackets, like this: ['this','is',1,'list']

Just like with strings, you can use the index (position number) of an item in your list. Try adding this to the Trinket: print my_list[0]
