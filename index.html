To make our hangman game, we are going to use a library called turtle, which comes with Python.

Drawing with turtle is pretty easy. The idea is that the 'turtle' runs around in the window, leaving a trail behind it. You, the programmer, can tell it what direction and how far to go.

Try this in the Trinket window:
import turtle # we need to import libraries we want to use
my_turtle = turtle.Turtle() # this makes a new turtle called "my_turtle"
my_turtle.forward(100)
my_turtle.right(45)
my_turtle.forward(50)
my_turtle.left(90)
my_turtle.forward(20)

You can move the turtle without drawing if you first use this command
my_turtle.penup()

then, to start drawing again, use:
my_turtle.pendown()

There are other commands that can change the color and the thickness of the line, make circles, etc.

Complete documentation can be found here:

docs.python.org/2/library/turtle.html

Try cutting and pasting the code above in this trinket...then change the numbers and rerun to see what the changes do.
